# 💧 Water Potability Prediction

## 📌 Project Overview

This project applies machine learning techniques to predict whether water is safe for human consumption based on its physicochemical properties.

The analysis explores the Water Potability dataset and builds predictive models using features such as pH, hardness, solids, chloramines, sulfate concentration, conductivity, organic carbon, trihalomethanes, and turbidity.

The goal is to develop a data-driven approach to water quality assessment and environmental monitoring.

---

## 🎯 Objectives

* Perform exploratory data analysis (EDA) on water quality data
* Handle missing values and data imbalance
* Visualize feature distributions and relationships
* Build a classification model to predict water potability
* Evaluate model performance

---

## 📊 Dataset Information

* **Source:** Kaggle Water Potability Dataset
* **Total Samples:** 3276
* **Features:** 9 water quality indicators
* **Target Variable:** Potability (0 = Not potable, 1 = Potable)

### Features Used

* pH
* Hardness
* Total Dissolved Solids
* Chloramines
* Sulfate
* Conductivity
* Organic Carbon
* Trihalomethanes
* Turbidity

---

## 🔍 Exploratory Data Analysis

The project includes:

* Class distribution analysis
* Feature distribution visualization
* Missing value analysis
* Statistical summary of variables

Key insight: The dataset is imbalanced, with more non-potable water samples than potable samples.
